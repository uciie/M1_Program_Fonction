## Les codes en Java
Où se situer
```
M1_Program_Fonction/java
```

Les commandes
```
./gradlew build // compilation
./gradlew run // execution
```
si on veut lancer une classe spécifique avec java -cp :
```
./gradlew build // compilation
java -cp build/classes/java/main com.example.MonFichier // execution
```
